Brickpresso

A simple preso tool,
built with Mozilla Brick.

Slides are x-cards

With optional header elements.

Slide navigation

Right arrow, left arrow.

Or: tap on the right half, tap on the left half.

Fullscreen support

Using Fullscreen API. Press 'F' to toggle.

Stores slide # in window.hash

Reloading slides is safe.

It can be printed!

Want to generate a PDF?

Simply use your browser's Print or Print as PDF... functionalities.

For best results, ensure you configure the output to display no headers, and print in landscape (closer to the window aspect ratio).

If you use background images, set it to print backgrounds too!
